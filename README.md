# Web Development Final Project

This repository contains the **final project** for the Web Development course. It demonstrates practical skills in **HTML, CSS, JavaScript**, and **responsive web design**.

---

## Project Overview

The project showcases:

- **Responsive Layouts** – Works on desktop, tablet, and mobile devices.
- **Interactive Elements** – Buttons, forms, and navigation menus with JavaScript functionality.
- **Styled Components** – Custom CSS for visually appealing design.
- **Clean Code Structure** – Organized files and folders for easy maintenance.

---

## Features

- Fully responsive web pages
- Navigation bar with active link highlighting
- Form validation using JavaScript
- Styled buttons, cards, and sections
- Footer with contact information
